Ready to Connect?

We support both Java and Bedrock cross-play connections seamlessly.

Java Edition

PC / MAC Client

  1. Launch Minecraft Java 1.21+.
  2. Go to Multiplayer > Add Server.
  3. Enter server name and IP: play.theerpsmp.net.
  4. Save, click join, and enjoy!

Bedrock Edition

Mobile / Console / Windows 10

  1. Open Minecraft Bedrock.
  2. Go to Servers tab > Scroll down > click Add Server.
  3. Server IP: play.theerpsmp.net.
  4. Port: 19132 (Default).
  5. Click Save & Join!

Server Commands

A quick reference list of available player command interfaces.

/register <password> <confirmPassword> Register a password for your account. Required before you can play.
/login <password> Log in to your account with your registered password.
/shop Open the main player shop to trade items using Erpies.
/sell Open the quick sell inventory to convert items in hand directly to Erpies.
/derpshop Open the special Derpshop GUI to purchase keys and spawners using Derpies.
/pay <amount> <player> Transfer a specified amount of Erpies directly to another online player.
/bank Open your bank account interface to deposit or withdraw items and money.
/auction Open the global Auction House menu to buy or sell player listings.
/orders Open the buy orders board to request custom items or view pending requests.
/bh Open the Bounty Hunter head trading menu to exchange player heads for money.
/store Show the official store web page link to browse and buy passes.
/spawn Teleport safely back to the spawn protection zone.
/home Open the Homes Menu to teleport to one of your saved home locations.
/sethome Open the Homes Menu to save or manage your home points.
/rtp [overworld|nether|end] Randomly teleport within the Overworld, Nether, or End dimensions.
/tpa <player> Send a teleport request to another online player.
/tpaccept Accept a pending teleport request.
/warp <name> Teleport directly to any public server warp point.
/afk Teleport to the server's AFK zone to earn Derpies automatically over time.
/maketeam <name> Create a new player team/clan on the server.
/team Open your active team management menu to view members, vault, and settings.
/requesteam <player> Send a join request to another player's team.
/teamaccept Accept a pending team invitation request.
/dual <player> Send a direct duel invitation to another player in the Bedrock arena.
/dualaccept Accept a pending direct duel challenge within the 60-second window.
/apocalypse Open the Apocalypse mini-game menu to challenge waves of zombies.
/nametag Open the nametag menu to manage and equip your floating name tags.
/offhand Swap items between your main hand and offhand (useful for Bedrock players).
/setting Open the settings menu to configure chat spam, tpa requests, and other features.
/rules Open the server rules book UI to read the server guidelines.
/credits Open the server credits book UI to view developers and contributors.
